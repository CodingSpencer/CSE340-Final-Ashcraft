## Roles

### Admin
- Full access to all features
- Can view and manage all rentals
- Can delete any review
- Can manage users and vehicles
- Can view contact messages

### Employee
- Can view and manage all rentals
- Can update rental statuses
- Can view customer information
- Can view reviews

### Customer
- Can browse and view vehicle details
- Can submit rental requests
- Can add reviews to vehicles (one review per vehicle)
- Can edit and delete their own reviews
- Can view their own rentals
- Can submit contact messages

### Guest
- Can browse inventory
- Can view vehicle details
- Cannot submit rentals or reviews
- Can submit contact messages
